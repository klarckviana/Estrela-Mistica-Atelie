# Estrela Mística Ateliê

## Escopo Fechado

O projeto Estrela Mística Ateliê foi desenvolvido para apresentar as coleções e os serviços do ateliê, permitindo que clientes conheçam os produtos e entrem em contato para solicitar peças sob medida.

**Público-alvo:** pessoas interessadas em vestimentas religiosas, clientes do ateliê e praticantes de religiões de matriz africana.

**Paleta de cores:**
- Azul-marinho (#0B1033)
- Dourado (#D4AF37)
- Branco (#FFFFFF)

Essas cores foram escolhidas por transmitirem elegância, sofisticação e identidade visual da marca.

**Tipografia:**
Foram utilizadas fontes elegantes para representar a identidade do ateliê, combinadas com uma fonte de fácil leitura para os textos.

**Framework:**
Não foi utilizado framework. O projeto foi desenvolvido utilizando HTML5, CSS3 e JavaScript.

---

## Detalhamento do Site

O site possui seis páginas:

- **Início:** apresentação da empresa, destaques e acesso rápido.
- **Sobre:** história do ateliê, missão, visão e valores.
- **Coleções:** apresentação das linhas de vestimentas.
- **Galeria:** fotos dos produtos confeccionados.
- **Pedido:** formulário para solicitar orçamento.
- **Contato:** informações e formulário de contato.

Estrutura utilizada:

- HTML5
- CSS3
- JavaScript
- Organização em pastas CSS, JS e imagem.
